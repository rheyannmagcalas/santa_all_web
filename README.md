# Santa All Web

POC Web for Santa All Recommender Engine

## Installation


```bash
create virtual environment
pip install -r requirements.txt
python manage.py migrate 
python manage.py createsuperuser 
```

## Usage

```python
python manage.py runserver 0:8000
```

## Technologies

Django
SQLite
